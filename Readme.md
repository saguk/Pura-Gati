IPS Skater adalah game berbasis web (browser game) yang menggabungkan aksi skateboard seru dengan kuis materi Ilmu Pengetahuan Sosial (IPS). Game ini dirancang untuk membuat proses belajar menjadi lebih menyenangkan (Gamification) bagi siswa SMP/MTs.

Pemain akan mengendalikan karakter skater, menghindari rintangan, dan mengumpulkan "Gulungan Ilmu" untuk menjawab pertanyaan-pertanyaan IPS.
